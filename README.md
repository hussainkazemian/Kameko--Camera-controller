# MediaPipe Hand Gesture with Fullscreen

Tämä projekti käyttää MediaPipe Tasks Vision -kirjastoa käsieleiden tunnistukseen web-kameran kuvasta.

- Kameran kuva ja piirtoalue (canvas) täyttävät koko näytön.
- Toteutus on tehty JavaScriptillä ja MediaPipe Tasks Vision -kirjastolla.
- Tyylit on määritelty suoraan `index.html`-tiedostossa, ei erillisessä CSS-tiedostossa.

## Käynnistys

1. Avaa projekti selaimessa (esim. Vite dev-serverillä).
2. Salli kameran käyttö.
3. Näet kameran kuvan ja tunnistetut käsieleet koko näytöllä.

## Tiedostot

- `index.html`: sisältää tyylit ja rakenteen
- `src/main.js`: käsittelee kameran ja eleiden tunnistuksen
